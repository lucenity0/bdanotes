Summary
This technical guide provides a practical overview of managing data within
Cassandra, a distributed NoSQL database, by utilizing the CQL shell interface. It
outlines the foundational hierarchy of the system, starting with the creation of a
Keyspace—a container for data—and moving into the definition of Tables with
specific primary keys. The text details essential CRUD operations for manipulating
records, while highlighting specialized features like Secondary Indexes for non-
primary key queries and Collection types for storing sets and lists. Finally, it covers
advanced utility functions such as Counters, Time To Live (TTL) for temporary
data, and methods for importing or exporting datasets via CSV files.

Keywords: Apache Cassandra Commands, Keyspace and Tables, CRUD Operations,
Data Collections, Import and Export
